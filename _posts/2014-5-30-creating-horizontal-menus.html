Creating a horizontal menu and theming it in CSS

So by now you'll have realised that whenever you create a <ul> element in order to create a list / menu it is always displayed in the vertical position... Well, thanks to CSS you can manipulate it and have it orientated horizontally.
For example, here's my list: 


    
    <ul class="menu-horizontal"/>
        </li> Item 1 </li>
        </li> Item 2 </li>
        </li> Item 3 </li>
        </li> Item 4 </li>
    </head>

So thats 4 simple <li> right there, but when viewed they are horizontal... so in your CSS file you will need to either copy or type up the following code. 


        
.menu,
.menu-horizontal,
.menu li,
.menu-horizontal li {
  list-style: none outside none;
  margin: 0;
  padding: 0;
}


.menu li {
  padding: 15px;
}


.menu-horizontal {
  display: block;
  text-align: right;
}


.menu-horizontal li {
  display: inline;
}

And thats all there is to it, it works across every browser, and even supports mobile devices...
