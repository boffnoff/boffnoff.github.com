Using JavaScript's InnerHTML feature

In case you were never aware, JS has a useful function that enable you to dynamically change the content on a page without actually changing pages or reloading the current page. This could be useful as it enables quick refresh of text and can be carried out by the user. For example, in an App, why would you want to create seperate pages and lots of lines of code, when you could just use several JS buttons that change the content??



Anyway, here's an example of it at work...

Hello John Smith!

As you can see from above the text displayed changed dynamically... And without the page refreshing, and it doesnt have to be displayed in bold, I just chose to display it as such so you could see a more noticable change.
As you will be wondering how you achieve such perfection I have, of course, included the code for the example below for you! 


        
.menu,
.menu-horizontal,
.menu li,
.menu-horizontal li {
  list-style: none outside none;
  margin: 0;
  padding: 0;
}


.menu li {
  padding: 15px;
}


.menu-horizontal {
  display: block;
  text-align: right;
}


.menu-horizontal li {
  display: inline;
}

And thats all there is to it, it works across every browser, and even supports mobile devices...
