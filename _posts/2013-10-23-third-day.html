However, Dave has found a simpler way of doing the exact same thing. It is still within the "post.html" file, but the following code is underneath that above. Observe: 


<ul class="paging"> 
        {% if page.previous.url %} 
        <li class="paging__previous"> 
                <a href="{{page.previous.url}}">&laquo; {{page.previous.title}}</a> 
        </li> 
        {% endif %} 
        {% if page.next.url %} 
        <li class="paging__next"> 
                <a href="{{page.next.url}}">{{page.next.title}} &raquo; </a> 
        </li> 
        {% endif %} 
</ul>
